Descripción de terminos

Función
Una función es un bloque de código el cuál realiza alguna operación.
Let
Es una variable de JS que nos permite modificar su valor, sin embargo estas no se pueden volver a declarar.
Const
Es una variable de JS que mantiene un valor fijo el cual no puede ser modificado, ni tampoco puede volver a declararse.
Img
la etiqueta "< img >" nos permite mostrar imagenes.
A
la etiqueta "< a >" nos permite crear enlaces a otras paginas.
Span
La etiqueta "< span >" nos permite personalizar el estilo de solo una parte del texto.
